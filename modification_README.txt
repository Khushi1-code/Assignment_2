-------------GAME MODIFICATIONS----------

This version of the game includes three enhancements:

1. Start Menu System

A menu appears at the start allowing the player to Start Game or Exit.

After a game ends, the player can return to the menu to play again.

2. Score System

Player earns 1 point each time an obstacle passes without collision.

Current score is displayed during gameplay.

3. Player Lives

Player starts with 3 lives.

Colliding with an obstacle reduces lives by 1.

Game ends only when all lives are lost.
